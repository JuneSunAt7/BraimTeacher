# Решение задачи Braim Основной тур кейса "С++: Код решает всё"
 Командам предстоит разработать приложение, которое позволит быстро и
удобно работать с распределением учебной нагрузки между преподавателями
и формирования документов для кафедры.
Разрабатываемый сервис должен обеспечивать удобный импорт данных,
возможность добавлять в систему новых преподавателей, возможность
редактирования и корректировки информации по нагрузке на каждого их, а
также автоматическую генерацию документов: нагрузки преподавателей,
сведений к расписанию и индивидуальных планов преподавателей.
